# 作文教学 RAG 混合架构系统

## 项目简介
这是一个基于 RAG（Retrieval-Augmented Generation）架构的作文教学辅助系统，能够为作文题目提供智能化的教学指导和写作建议。

## 功能特性
- 作文题目解析和理解
- 相关素材检索
- 写作思路生成
- 范文参考推荐
- 个性化写作指导

## 技术架构
- **检索模块**: 基于向量数据库的语义检索
- **生成模块**: 结合大语言模型的内容生成
- **知识库**: 作文素材、范文、写作技巧库
- **混合策略**: 结合关键词检索和语义检索

## 项目结构
```
article-rag/
├── src/
│   ├── core/              # 核心模块
│   ├── retrieval/         # 检索模块
│   ├── generation/        # 生成模块
│   ├── knowledge/         # 知识库管理
│   └── api/              # API接口
├── data/                 # 数据存储
├── config/              # 配置文件
├── tests/               # 测试文件
└── requirements.txt     # 依赖包
```

## 快速开始
1. 安装依赖: `pip install -r requirements.txt`
2. 配置环境变量
3. 初始化知识库
4. 运行服务
